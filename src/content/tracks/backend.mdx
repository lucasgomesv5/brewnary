A engenharia pesada. Aqui você aprende a construir sistemas robustos que sustentam milhares de usuários.

## O que você vai aprender

- **APIs** — REST design, gRPC/tRPC, GraphQL, WebSockets, API design avançado (pagination, rate limiting, idempotency, webhooks)
- **Node.js Internals** — Runtime, V8, libuv, Event Loop, Streams, Cluster e Worker Threads
- **Linguagens** — Go, Java e Rust (ownership, borrowing, async, Axum)
- **Bancos de Dados** — SQL, índices, performance, transações, NoSQL, PostgreSQL internals (MVCC, vacuum, pgvector)
- **Event-Driven** — Kafka internals, Event Sourcing, CQRS, Outbox pattern
- **Serverless** — AWS Lambda, Step Functions, cold start optimization, SST/CDK
- **Segurança** — JWT, OAuth, OWASP Top 10, criptografia aplicada
- **Testes** — Pirâmide de testes, contract testing, mutation testing
- **Escalabilidade** — Redis (distributed locking, Streams), message queues, sagas, feature flags
- **Arquitetura Backend** — Rate limiting, circuit breaker, bulkhead pattern

## Pré-requisitos

Ter completado a trilha de Ciência da Computação, especialmente Estruturas de Dados e Redes.

## Tempo estimado

**3-4 meses** estudando 1-2 horas por dia.
