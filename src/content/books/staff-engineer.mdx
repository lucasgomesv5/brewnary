## Por que ler

O que vem depois de Senior? Este livro mapeia os papeis de Staff, Principal e Distinguished Engineer. Se seu objetivo e crescer na trilha tecnica (nao gestao), este e o guia.

## Conceitos-chave

**4 Arquetipos de Staff Engineer:**
- **Tech Lead:** Guia a direcao tecnica de um time
- **Architect:** Define a arquitetura entre times/sistemas
- **Solver:** Resolve os problemas mais dificeis onde quer que surjam
- **Right Hand:** Braco direito de um VP/CTO, executando prioridades estrategicas

**O que muda de Senior para Staff:**
```
Senior:                    Staff:
Escopo: 1 time             Escopo: múltiplos times
Resolve problemas          Define QUAIS problemas resolver
Implementa                 Influencia via documentos (RFC, ADR)
Código é o output          Impacto organizacional é o output
Profundidade técnica       Profundidade + amplitude
```
